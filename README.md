# Инструкция

Приложение для чтения сгенерированных .js файлов и последующей конвертации в .docx файлы

## Установка

1. Клонировать репозиторий
2. Установить NodeJS (https://nodejs.org/)
3. Для проверки работы выполнить:

```
node -v
npm -v
```

4. Переместиться в корень проекта
5. Выполнить установку библиотек:

```
npm install
```

## Запуск

Чтобы выполнить запуск, нужно выполнить:

```
npm start
```

## Использование

При запуске пользователю доступен список опций (Все или конкретная).
Чтобы выбирать, нужно воспользоваться стрелочками вверх-вниз.
Файлы для генерации выбираются из папки `/script`, а результат помещается в папку `/docs`.
Для того, чтобы добавить новый скрипт, нужно добавить его в папку `/script`.
Стоит отметить, что нейросеть выдаёт результат со случайным путём сохранения,
поэтому это надо или указать в промпте, или изменить самостоятельно после генерации
Обычно путь сохранения находится в конце файла.
Шаблонный скрипт используемый для генерации находится в файле `prompt.json`
