# Инструкция

Приложение для чтения сгенерированных .js файлов и последующей конвертации в .docx файлы

## Установка

1. Клонировать репозиторий
2. Переместиться в корень проекта
3. Выполнить установку библиотек:

```
npm install
```

## Запуск

Чтобы выполнить запуск, нужно выполнить:

```
npm start
```
