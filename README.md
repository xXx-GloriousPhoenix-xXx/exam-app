# Инструкция

Приложение для чтения сгенерированных .js файлов и последующей конвертации в .docx файлы

## Установка

1. Клонировать репозиторий
2. Установить NodeJS (https://nodejs.org/)
2.1 Для проверки работы выполнить:

```
node -v
npm -v
```


3. Переместиться в корень проекта
4. Выполнить установку библиотек:

```
npm install
```

## Запуск

Чтобы выполнить запуск, нужно выполнить:

```
npm start
```
